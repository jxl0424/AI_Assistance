This folder is your **Jarvis Knowledge Base**.

## How to use
1.  **Drop your files here**:
    *   PDFs (`.pdf`)
    *   Text files (`.txt`)
    *   Markdown files (`.md`)
    *   Word docs (`.docx`)

2.  **Ask Jarvis**:
    *   "Summarize the [Filename] document."
    *   "What does my knowledge base say about [Topic]?"
    *   "Search my notes for [Keyword]."

Jarvis will automatically index these files when you start the system or run the update command.
